# TFG_2526_GEIA_27
Diseño de tarjeta de entrenamiento y evaluación basada en microcontrolador de bajo coste.

El equipo diseñado integrará la periferia necesaria para evaluar entradas/salidas digitales y analógicas, 
buses de comunicación (UART, I2C, SPI), y comunicación inalámbrica (WiFi y/o Bluetooth).
